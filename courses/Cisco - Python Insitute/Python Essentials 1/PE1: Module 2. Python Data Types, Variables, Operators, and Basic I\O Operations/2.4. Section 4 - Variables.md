## What is a variable?

A named container that stores a value. Created automatically when you assign a value to it — no declaration needed.

```python
var = 1          # creates variable and assigns 1
print(var)       # 1
var = "hello"    # reassign — any type allowed
print(var)       # hello
```

---

## Naming rules

| Rule | Example |
|---|---|
| Letters, digits, `_` only | `my_var`, `x1`, `_temp` |
| Must start with a letter or `_` | ✅ `var1` ❌ `1var` |
| Case-sensitive | `var` ≠ `Var` ≠ `VAR` |
| Cannot be a keyword | ❌ `for`, `if`, `True`, `import` |

**PEP 8 convention:** use `snake_case` for variables and functions.

```python
# ✅ Good
my_variable = 10
days_to_deadline = 5

# ❌ Bad
10t = 5          # starts with digit
my variable = 5  # contains space
del = 5          # reserved keyword
```

### Python keywords (cannot be used as names)
```
False, None, True, and, as, assert, break, class, continue,
def, del, elif, else, except, finally, for, from, global,
if, import, in, is, lambda, nonlocal, not, or, pass, raise,
return, try, while, with, yield
```

## Assignment operator

`=` assigns the value on the right to the variable on the left.

```python
x = 10
y = x + 5       # use variable in expression
z = (x ** 2 + y ** 2) ** 0.5   # Pythagorean theorem
print("c =", z)
```

---

## Shortcut operators

| Shortcut | Equivalent |
|---|---|
| `x += 2` | `x = x + 2` |
| `x -= 2` | `x = x - 2` |
| `x *= 2` | `x = x * 2` |
| `x /= 2` | `x = x / 2` |
| `x //= 2` | `x = x // 2` |
| `x %= 2` | `x = x % 2` |
| `x **= 2` | `x = x ** 2` |

```python
x = 10
x += 5     # x is now 15
x *= 2     # x is now 30
x //= 4    # x is now 7
print(x)   # 7
```

## Combining strings and variables

```python
version = "3.11"
print("Python version: " + version)   # concatenation

name = "Ruben"
age = 28
print("Name:", name, "Age:", age)     # multiple args to print
```

## Exercises

### Exercise 1 — Variable assignment
Create variables `a = 3.0` and `b = 4.0`, calculate the hypotenuse `c` and print `"c = "` followed by the result.

<details>
<summary>Solution</summary>

```python
a = 3.0
b = 4.0
c = (a ** 2 + b ** 2) ** 0.5
print("c =", c)   # c = 5.0
```
</details>

### Exercise 2 — Shortcut operators
Start with `x = 100`. Using only shortcut operators: add 50, multiply by 2, divide by 5, subtract 10. Print the result.

<details>
<summary>Solution</summary>

```python
x = 100
x += 50    # 150
x *= 2     # 300
x /= 5     # 60.0
x -= 10    # 50.0
print(x)   # 50.0
```
</details>

### Exercise 3 — Miles to kilometers
`1 mile ≈ 1.61 km`. Convert `7.38 miles` to km and `12.25 km` to miles. Use `round()` to 2 decimal places.

<details>
<summary>Solution</summary>

```python
miles = 7.38
km = 12.25

print(miles, "miles is", round(miles * 1.61, 2), "kilometers")
print(km, "kilometers is", round(km / 1.61, 2), "miles")
```
</details>
