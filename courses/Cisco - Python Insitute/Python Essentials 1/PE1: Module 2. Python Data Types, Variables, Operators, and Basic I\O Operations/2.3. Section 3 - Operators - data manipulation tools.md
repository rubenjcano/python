## Arithmetic operators

| Operator | Name | Example | Result |
|---|---|---|---|
| `+` | Addition | `2 + 3` | `5` |
| `-` | Subtraction | `5 - 2` | `3` |
| `*` | Multiplication | `3 * 4` | `12` |
| `/` | Division | `7 / 2` | `3.5` |
| `//` | Floor division | `7 // 2` | `3` |
| `%` | Modulo (remainder) | `7 % 2` | `1` |
| `**` | Exponentiation | `2 ** 3` | `8` |

---

## Int vs Float rules

- `/` **always** returns a float — even `4 / 2 = 2.0`
- `//` rounds **down** to the nearest integer — always toward the lesser value
- `**`, `*`, `+`, `-` return int if both operands are int, float if either is float

```python
print(7 / 2)     # 3.5   ← always float
print(7 // 2)    # 3     ← floor division
print(7.0 // 2)  # 3.0   ← float floor division
print(-7 // 2)   # -4    ← rounds DOWN (not toward zero!)
print(2 ** 3)    # 8     ← int
print(2 ** 3.0)  # 8.0   ← float
```

---

## Modulo

Returns the **remainder** after integer division.

```python
print(14 % 4)    # 2  → 14 = 3×4 + 2
print(12 % 4.5)  # 3.0
print(5 % 2)     # 1
```

> ⚠️ Never divide (or modulo) by zero — causes `ZeroDivisionError`.

---

## Unary vs Binary operators

- **Binary** — two operands: `4 + 5`, `10 - 3`
- **Unary** — one operand: `-3` (negation), `+3` (no effect)

```python
print(-3)    # -3
print(+3)    # 3
print(-(-3)) # 3
```

---

## Operator priority

From highest to lowest:

| Priority | Operator | Notes |
|---|---|---|
| 1 | `**` | Right-sided binding |
| 2 | unary `+`, `-` | |
| 3 | `*`, `/`, `//`, `%` | Left-sided binding |
| 4 | `+`, `-` | Left-sided binding |

```python
print(2 + 3 * 5)      # 17 — multiplication first
print(2 ** 2 ** 3)    # 256 — right-sided: 2 ** (2**3) = 2**8
print(-3 ** 2)        # -9  — ** before unary minus: -(3**2)
print(9 % 6 % 2)      # 1   — left-sided: (9%6) % 2 = 3 % 2
```

> Use **parentheses** to override priority and improve readability.

```python
print((2 + 3) * 5)    # 25
print(-(3 ** 2))      # -9
```

## Exercises

### Exercise 1 — Predict the output
```python
print(2 * 3 % 5)
print(2 ** 2 ** 3)
print(-2 ** 3)
```

<details>
<summary>Solution</summary>

```
1   → (2*3) % 5 = 6 % 5 = 1  (left-sided, same priority)
256 → 2 ** (2**3) = 2**8     (right-sided binding)
-8  → -(2**3) = -8            (** before unary minus)
```
</details>

### Exercise 2 — Floor division with negatives
```python
print(-7 // 2)
print(7 // -2)
print(-7 // -2)
```

<details>
<summary>Solution</summary>

```
-4  → real result -3.5, rounded DOWN to -4
-4  → real result -3.5, rounded DOWN to -4
3   → real result 3.5, rounded DOWN to 3
```
</details>

---

### Exercise 3 — Mixed types
```python
print(2 ** 4)
print(2 * 4.)
print(7 / 2)
print(7 // 2)
print(7 // 2.)
```

<details>
<summary>Solution</summary>

```
16    → int ** int = int
8.0   → int * float = float
3.5   → / always float
3     → int // int = int
3.0   → int // float = float
```
</details>
