## What is a literal?

A literal is data whose value is determined by itself — it doesn't need context to be understood.

```python
print("Hello")   # "Hello" is a string literal
print(123)       # 123 is an integer literal
```

---

## Integers

Whole numbers — no fractional part.

```python
print(123)          # positive
print(-11111111)    # negative
print(11_111_111)   # underscores for readability (Python 3.6+)
print(+11)          # plus sign is optional
```

### Octal and hexadecimal

```python
print(0o123)   # octal → 83 in decimal
print(0x123)   # hexadecimal → 291 in decimal
```

---

## Floats

Numbers with a fractional part — the `.` makes it a float.

```python
print(2.5)      # float
print(-0.4)     # float
print(4.)       # same as 4.0
print(.4)       # same as 0.4
print(4)        # integer — no dot!
```

### Scientific notation

```python
print(3E8)        # 300000000.0 (speed of light)
print(6.62607E-34)  # Planck's constant
print(0.0000000000000000000001)  # → Python prints: 1e-22
```

> Python always uses the most compact notation when displaying floats.

---

## Strings

Text delimited by quotes — `"..."` or `'...'`.

```python
print("I like \"Monty Python\"")   # escaped quotes
print('I like "Monty Python"')     # single quotes — no escaping needed
print("I'm Monty Python.")         # apostrophe inside double quotes
print('I\'m Monty Python.')        # escaped apostrophe
print("")                          # empty string — still valid
```

---

## Booleans

Two values only: `True` and `False` — case-sensitive.

```python
print(True)    # True
print(False)   # False
print(True > False)   # True  (1 > 0)
print(True < False)   # False (1 < 0)
```

> In numeric context: `True = 1`, `False = 0`

---

## None

A special literal representing the **absence of a value**.

```python
print(None)   # None
```

---

## Exercises

### Exercise 1 — Identify the type
What type is each of these literals?
```python
"1.5"    2.0    528    False    None    0o17    "True"
```

<details>
<summary>Solution</summary>

```
"1.5"  → str
2.0    → float
528    → int
False  → bool
None   → NoneType
0o17   → int (octal → 15 in decimal)
"True" → str (it's in quotes!)
```
</details>

---

### Exercise 2 — Escape characters in strings
Print this output using a **single** `print()`:
```
"I'm"
""learning""
"""Python"""
```

<details>
<summary>Solution</summary>

```python
print("\"I'm\"\n\"\"learning\"\"\n\"\"\"Python\"\"\"")
```
</details>

### Exercise 3 — Predict the output
What does this print?
```python
print(True > False)
print(True < False)
print(True + True)
print(False + False)
```

<details>
<summary>Solution</summary>

```
True    (1 > 0)
False   (1 < 0)
2       (1 + 1)
0       (0 + 0)
```
</details>
