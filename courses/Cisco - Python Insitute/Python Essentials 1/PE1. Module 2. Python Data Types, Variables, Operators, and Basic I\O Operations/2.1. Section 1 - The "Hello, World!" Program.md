## The print() function

`print()` is a built-in function that outputs text to the console. Always needs parentheses.

```python
print("Hello, World!")    # basic usage
print()                   # prints a blank line
print("a", "b", "c")     # multiple arguments → a b c
```

**Functions can come from:**
- Python itself (built-in) — like `print()`
- Modules — need to be imported
- Your own code

---

## Strings

Text delimited by quotes — everything inside is treated as data, not code.

```python
print("double quotes")
print('single quotes')
print("It's fine")           # single quote inside double ✅
print('She said "hi"')       # double quote inside single ✅
print("She said \"hi\"")     # escaped quote ✅
```

---

## Escape characters

| Sequence | Meaning |
|---|---|
| `\n` | New line |
| `\t` | Tab |
| `\\` | Literal backslash |
| `\'` / `\"` | Literal quote |

```python
print("Line 1\nLine 2")      # two lines
print("C:\\Users\\Ruben")    # backslash in path
```

---

## Keyword arguments

Change `print()` default behavior. Must come **after** positional arguments.

```python
# end — replaces the default newline
print("Hello", end=" ")
print("World")               # → Hello World

# sep — replaces the default space between arguments
print("2025", "01", "15", sep="-")   # → 2025-01-15

# combining both
print("A", "B", "C", sep="*", end="!\n")  # → A*B*C!
```

### Exercise 1 — sep and end
Produce this output using two `print()` calls:
```
Programming***Essentials***in...Python
```

<details>
<summary>Solution</summary>

```python

```

Print this using a single `print()`:
