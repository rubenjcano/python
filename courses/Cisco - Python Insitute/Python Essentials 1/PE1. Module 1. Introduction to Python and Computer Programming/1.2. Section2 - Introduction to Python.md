## What is Python?

A widely-used, interpreted, object-oriented, high-level programming language with dynamic semantics, used for general-purpose programming. Named after **Monty Python's Flying Circus** (BBC comedy series), not the snake.

---

## Who created it?

**Guido van Rossum** (born 1956, Netherlands) — started Python in December 1989 as a hobby project during Christmas. His goals for Python:

- Easy and intuitive, yet powerful
- Open source — anyone can contribute
- Code as readable as plain English
- Suitable for everyday tasks with short development times

All four goals have been fulfilled. Python consistently ranks in the **top 3 most popular languages** worldwide (PYPL, TIOBE indexes).

---

## Why Python?

- **Easy to learn** — shorter learning curve than most languages
- **Easy to read** — code is clean and close to natural language
- **Easy to write** — faster to develop new software
- **Free, open source, and multiplatform** — works everywhere
- **Huge community** — thousands of contributors and libraries

---

## Where is Python used?

- Search engines, cloud storage, social media platforms
- Data analysis and scientific computing
- Development tooling and automation
- Software testing and QA
- Everyday desktop and web applications

## Where Python is NOT ideal

- **Low-level / "close to metal" programming** — drivers, graphics engines (use C/C++)
- **Mobile applications** — still limited native support

---

## Python 2 vs. Python 3

| | Python 2 | Python 3 |
|---|---|---|
| Status | End of life — no new features | Active — current version |
| Compatibility | Not compatible with Python 3 | Not backward compatible with Python 2 |
| Use today | Only for maintaining legacy systems | Use this for all new projects |

> ✅ Always use **Python 3** for new projects — this is what this course uses.

Minor differences exist between Python 3 releases, but all newer versions are **backward compatible** with previous Python 3 versions.

---

## Python Implementations

| Implementation | Language it's written in | Key use case |
|---|---|---|
| **CPython** | C | The standard, reference implementation — what most people use |
| **Cython** | C | Translates Python to C for performance-heavy mathematical code |
| **Jython** | Java | Integrates Python into Java environments (follows Python 2) |
| **PyPy** | RPython (subset of Python) | Faster execution via JIT compilation — used for testing new Python features |
| **MicroPython** | C | Optimized Python 3 for microcontrollers and embedded hardware |

> 📌 This course focuses on **CPython** — the most widely used implementation.
