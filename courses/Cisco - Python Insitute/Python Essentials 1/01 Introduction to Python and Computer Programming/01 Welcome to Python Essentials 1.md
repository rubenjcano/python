# 🐍 Python Essentials 1 — Course Overview

> 📖 Based on: [Python Essentials 1 — OpenEDG Python Institute x Cisco Networking Academy](https://pythoninstitute.org/)
> Prepares for: [PCEP – Certified Entry-Level Python Programmer](https://pythoninstitute.org/certification/pcep-certification-entry-level/)

---

## What it is

Python Essentials 1 is the first course in a two-part series designed to take you from zero to a solid Python foundation. It covers the core concepts needed to design, write, run, debug, and improve Python programs — and fully prepares you for the PCEP certification.

---

## Who it's for

- Beginners wanting to learn programming for fun or work
- People targeting entry-level roles as software developer, data analyst, or tester
- Professionals exploring Python-connected technologies
- Managers wanting to understand the software development cycle

---

## What you will learn

- Universal concepts of computer programming
- Python syntax and semantics
- Practical skills for solving typical programming challenges
- The most important elements of the Python Standard Library
- How to set up your runtime environment
- How to design, develop, test, and debug simple Python programs

---

## 📚 Course Modules

### Module 1 — Introduction to Python and Computer Programming
How computers work, what programming is, and why Python is one of the most in-demand languages today.

### Module 2 — Data Types, Variables, Input/Output, and Basic Operators
The building blocks of every Python program: storing data in variables, reading user input, printing output, and performing operations.

### Module 3 — Boolean Values, Conditionals, Loops, Lists, and Bitwise Operations
Controlling the flow of a program with conditions and loops, working with collections of data, and understanding logical operations.

### Module 4 — Functions, Tuples, Dictionaries, Exceptions, and Data Processing
Writing reusable code with functions, working with more complex data structures, and handling errors gracefully.

---

## 🏅 PCEP Certification

**PCEP – Certified Entry-Level Python Programmer** is a professional credential from the Python Institute that validates:

- Understanding of Python syntax and semantics
- Proficiency with variables, operators, control flow, and functions
- Knowledge of Python's data type system
- Ability to handle exceptions and debug programs

**Why it matters:**
- Stands out on a CV for entry-level developer or data roles
- Interim step toward **PCAP – Certified Associate in Python Programming**
- Starting point for a career in software development and data engineering
