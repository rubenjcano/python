## Installing Python 3

| OS | Steps |
|---|---|
| **Linux** | Likely already installed. Open terminal and type `python3` to check. If missing, install via package manager (`python3`) |
| **Windows** | Download `.exe` from [python.org/downloads](https://www.python.org/downloads/) → run installer → ✅ check **"Add Python 3.x to PATH"** |
| **macOS** | Python 2 may be pre-installed but you need Python 3 → download `.pkg` from [python.org/downloads](https://www.python.org/downloads/) |

---

## IDLE — Your First Development Environment

Python 3 comes with **IDLE** (Integrated Development and Learning Environment) — a simple editor built for learning Python.

It has two windows:
- **Shell window** — where output appears and you can run quick commands
- **Editor window** — where you write and save your source code (`.py` files)

---

## Your First Program

1. Open IDLE → `File` → `New File`
2. Save it as `snake.py` (IDLE adds `.py` automatically)
3. Type this into the editor:

```python
print("Hisssssss...")
```

4. Run it: `Run` → `Run Module` (or press **F5**)
5. The shell window should display: `Hisssssss...`

> ⚠️ Always use **straight quotes** `"..."` — never curly/typographic quotes, Python won't accept them.

## Understanding Errors

When something goes wrong, Python shows an error message in the console. It includes:

| Part | What it tells you |
|---|---|
| **Traceback** | The path the code took before failing |
| **Location** | File name, line number, and module |
| **Erroneous line** | The content of the line that caused the error |
| **Error name** | The type of error and a short explanation |

> ⚠️ The reported line number may not be exactly where the bug is — Python reports where it *noticed* the problem, which can be slightly after the actual cause.

### Common error types

```python
# Missing closing parenthesis → SyntaxError (EOF)
print("Hisssssss..."

# Misspelled function name → NameError
prin("Hisssssss...")
```

---

## Key Takeaways

- Always check **"Add Python 3.x to PATH"** when installing on Windows
- IDLE has two windows — editor (write code) and shell (see output)
- Save files with `.py` extension
- Run code with **F5**
- Read error messages carefully — they tell you the type, location, and cause of the problem
