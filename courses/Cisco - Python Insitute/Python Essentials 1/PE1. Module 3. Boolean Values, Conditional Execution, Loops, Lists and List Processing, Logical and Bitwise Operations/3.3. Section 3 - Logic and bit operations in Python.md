## Logical operators

Used to combine conditions. Return `True` or `False`.

| Operator | Meaning | Example | Result |
|---|---|---|---|
| `and` | Both must be True | `True and False` | `False` |
| `or` | At least one must be True | `True or False` | `True` |
| `not` | Negates the value | `not True` | `False` |

```python
x = 5
print(x > 0 and x < 10)   # True  — both conditions met
print(x < 0 or x > 3)     # True  — one condition met
print(not x == 5)          # False — x is 5, negated
```

### Priority (low to high): `not` > `and` > `or`

### De Morgan's Laws
```python
not (p and q) == (not p) or (not q)
not (p or q)  == (not p) and (not q)
```

## Bitwise operators

Operate on **individual bits** of integers. Cannot be used with floats.

| Operator | Name | Example | Result |
|---|---|---|---|
| `&` | AND | `15 & 22` | `6` |
| `\|` | OR | `15 \| 22` | `31` |
| `~` | NOT (negation) | `~15` | `-16` |
| `^` | XOR | `15 ^ 22` | `25` |
| `<<` | Left shift | `17 << 2` | `68` |
| `>>` | Right shift | `17 >> 1` | `8` |

```python
x = 15   # 0000 1111
y = 16   # 0001 0000

print(x & y)   # 0  → 0000 0000
print(x | y)   # 31 → 0001 1111
print(x ^ y)   # 31 → 0001 1111
print(~x)      # -16 (two's complement)
print(y >> 1)  # 8  → divide by 2
print(y << 3)  # 128 → multiply by 8
```

### Shifts = fast multiply/divide by powers of 2
```python
n << 1  # n × 2
n << 2  # n × 4
n >> 1  # n // 2
n >> 2  # n // 4
```

### Shortcut operators
```python
x &= mask    # x = x & mask
x |= mask    # x = x | mask
x ^= mask    # x = x ^ mask
x <<= 2      # x = x << 2
x >>= 1      # x = x >> 1
```

## Bit masks — practical use

Check, set, reset, or toggle a specific bit using a mask.

```python
flag = 0b00001111   # some flags register
mask = 0b00001000   # target bit 3

# Check if bit is set
if flag & mask:
    print("bit is ON")

# Set the bit
flag |= mask

# Reset the bit
flag &= ~mask

# Toggle the bit
flag ^= mask
```

---

## Logical vs Bitwise — key difference

```python
i = 15
j = 22

print(i and j)   # True  — logical: both non-zero = True
print(i & j)     # 6     — bitwise: operates on each bit
```

### Exercise 1 — Predict the output
```python
x = 1
y = 0
z = ((x == y) and (x == y)) or not(x == y)
print(not(z))
```

<details>
<summary>Solution</summary>

```
False
```
`x == y` → `1 == 0` → `False`. So `z = (False and False) or not False` = `False or True` = `True`. Then `not True` = `False`.

### Exercise 2 — Bitwise operations
```python
x = 4   # 0100
y = 1   # 0001

a = x & y    # AND
b = x | y    # OR
c = ~x       # NOT
d = x ^ 5   # XOR
e = x >> 2   # right shift
f = x << 2   # left shift

print(a, b, c, d, e, f)
```

<details>
<summary>Solution</summary>

```
0 5 -5 1 1 16
```

| Var | Operation | Binary | Result |
|---|---|---|---|
| `a` | `4 & 1` | `0100 & 0001` | `0` |
| `b` | `4 \| 1` | `0100 \| 0001` | `5` |
| `c` | `~4` | two's complement of 4 | `-5` |
| `d` | `4 ^ 5` | `0100 ^ 0101` | `1` |
| `e` | `4 >> 2` | shift right 2 = 4//4 | `1` |
| `f` | `4 << 2` | shift left 2 = 4×4 | `16` |
</details>
