## What is Bubble Sort?

A simple sorting algorithm that repeatedly compares adjacent elements and swaps them if they're in the wrong order. The largest value "bubbles up" to the end after each pass.

**Not efficient for large lists** — but great for understanding how sorting works.

---

## How it works (step by step)

```
[8, 10, 6, 2, 4]
Pass 1: compare and swap adjacent pairs → largest (10) moves to end
Pass 2: second largest (8) moves to second-to-last
...until no swaps needed → list is sorted
```

---

## Implementation

### Single pass (one iteration)
```python
my_list = [8, 10, 6, 2, 4]

for i in range(len(my_list) - 1):
    if my_list[i] > my_list[i + 1]:
        my_list[i], my_list[i + 1] = my_list[i + 1], my_list[i]
```

### Full sort (repeat until no swaps)
```python
my_list = [8, 10, 6, 2, 4]
swapped = True

while swapped:
    swapped = False
    for i in range(len(my_list) - 1):
        if my_list[i] > my_list[i + 1]:
            swapped = True
            my_list[i], my_list[i + 1] = my_list[i + 1], my_list[i]

print(my_list)   # [2, 4, 6, 8, 10]
```

> The `swapped` flag stops the loop early if no swaps occurred — the list is already sorted.

---

## Python's built-in sort (use this in practice)

```python
lst = [8, 10, 6, 2, 4]
lst.sort()           # sort ascending in place
print(lst)           # [2, 4, 6, 8, 10]

lst.reverse()        # reverse in place
print(lst)           # [10, 8, 6, 4, 2]

lst.sort(reverse=True)  # sort descending
print(lst)           # [10, 8, 6, 4, 2]
```

> `sort()` and `reverse()` modify the list **in place** — they don't return a new list.

## Exercises

### Exercise 1 — Implement bubble sort
Sort `[5, 1, 3, 2, 4]` using bubble sort. Print after each pass to see the progress.

<details>
<summary>Solution</summary>

```python
my_list = [5, 1, 3, 2, 4]
swapped = True

while swapped:
    swapped = False
    for i in range(len(my_list) - 1):
        if my_list[i] > my_list[i + 1]:
            swapped = True
            my_list[i], my_list[i + 1] = my_list[i + 1], my_list[i]
    print(my_list)

print("Sorted:", my_list)
```
</details>

### Exercise 2 — Sort strings
Sort `["banana", "apple", "cherry", "date"]` alphabetically using `.sort()`, then reverse it.

<details>
<summary>Solution</summary>

```python
fruits = ["banana", "apple", "cherry", "date"]
fruits.sort()
print(fruits)    # ['apple', 'banana', 'cherry', 'date']

fruits.reverse()
print(fruits)    # ['date', 'cherry', 'banana', 'apple']
```
</details>
