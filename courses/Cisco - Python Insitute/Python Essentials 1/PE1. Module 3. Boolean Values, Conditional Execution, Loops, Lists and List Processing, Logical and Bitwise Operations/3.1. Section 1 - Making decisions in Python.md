## Comparison operators

| Operator | Meaning | Example | Result |
|---|---|---|---|
| `==` | Equal to | `2 == 2` | `True` |
| `!=` | Not equal to | `2 != 3` | `True` |
| `>` | Greater than | `5 > 3` | `True` |
| `>=` | Greater than or equal to | `5 >= 5` | `True` |
| `<` | Less than | `3 < 5` | `True` |
| `<=` | Less than or equal to | `3 <= 3` | `True` |

> ⚠️ `=` assigns a value. `==` compares values.

Comparison operators return `True` or `False` and can be stored in variables:

```python
result = 5 >= 3
print(result)   # True
```

---

## if statement

Executes a block only if the condition is `True`. The block must be **indented**.

```python
if x > 10:
    print("x is greater than 10")
print("this always runs")   # outside the if block
```

---

## if-else statement

Provides an alternative block when the condition is `False`.

```python
if x > 10:
    print("greater than 10")
else:
    print("10 or less")
```

---

## if-elif-else cascade

Checks multiple conditions in order — stops at the first `True`.

```python
if x > 100:
    print("large")
elif x > 50:
    print("medium")
elif x > 10:
    print("small")
else:
    print("tiny")
```

Rules:
- `else` is optional and must always be last
- Only **one** branch executes — the first one that is `True`
- If no condition is `True` and there is no `else`, nothing executes

---

## Nested if statements

An `if` inside another `if`. Each `else` pairs with the nearest `if` at the same indentation level.

```python
if weather_is_good:
    if restaurant_found:
        have_lunch()
    else:
        eat_sandwich()
else:
    go_to_theater()
```

## Finding the largest number

```python
# Two numbers
if number1 > number2:
    largest = number1
else:
    largest = number2

# Three numbers — compare iteratively
largest = number1
if number2 > largest:
    largest = number2
if number3 > largest:
    largest = number3

print("Largest:", largest)

# Built-in shortcut
print(max(number1, number2, number3))
print(min(number1, number2, number3))
```

## Leap year example

```python
year = int(input("Enter year: "))

if year < 1582:
    print("Not within the Gregorian calendar period")
elif year % 4 != 0:
    print("Common year")
elif year % 100 != 0:
    print("Leap year")
elif year % 400 != 0:
    print("Common year")
else:
    print("Leap year")
```

### Exercise 2 — if-elif-else
Read a string. Print:
- `"Yes - Spathiphyllum is the best plant ever!"` if input is `"Spathiphyllum"`
- `"No, I want a big Spathiphyllum!"` if input is `"spathiphyllum"`
- `"Spathiphyllum! Not {input}!"` otherwise

<details>
<summary>Solution</summary>

```python
plant = input("Enter plant name: ")

if plant == "Spathiphyllum":
    print("Yes - Spathiphyllum is the best plant ever!")
elif plant == "spathiphyllum":
    print("No, I want a big Spathiphyllum!")
else:
    print("Spathiphyllum! Not " + plant + "!")
```
</details>

### Exercise 3 — Nested if
Read three numbers and print the largest without using `max()`.

<details>
<summary>Solution</summary>

```python
a = int(input("First: "))
b = int(input("Second: "))
c = int(input("Third: "))

largest = a
if b > largest:
    largest = b
if c > largest:
    largest = c

print("Largest:", largest)
```

<details>
<summary>Q3 — x,y,z = z,y,x (swap) / print(x > z) / print((y-5) == x)</summary>

```
True
False
```
After swap: `x=8, y=10, z=5`. `8 > 5` is True. `(10-5) == 8` → `5 == 8` is False.
</details>

<details>
<summary>Q4 — x=10, three separate ifs with final else</summary>

```
True
True
else
```
All three `if`s are independent. `x < 10` is False so the `else` of the third `if` runs.
</details>

<details>
<summary>Q5 — x="1", nested if-elif-else + second if-else</summary>

```
four
five
```
`x == "1"` is True → enters elif chain → `int("1") > 1` False, `int("1") < 1` False → prints `"four"`. Then second `if`: `int("1") == 1` True → prints `"five"`.
</details>

<details>
<summary>Q6 — x=1, y=1.0, z="1"</summary>

```
one
two
```
