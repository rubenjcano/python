## What is a list?

An ordered, mutable collection of values. Elements are indexed starting from `0`.

```python
my_list = [10, 5, 7, 2, 1]
print(my_list[0])    # 10  — first element
print(my_list[-1])   # 1   — last element
print(my_list[-2])   # 2   — second to last
print(len(my_list))  # 5   — length
```

---

## Indexing and updating

```python
my_list = [10, 5, 7, 2, 1]
my_list[0] = 111          # update first element
my_list[1] = my_list[4]   # copy last to second
print(my_list)            # [111, 1, 7, 2, 1]
```

---

## Adding elements

```python
lst = [1, 2, 3]
lst.append(4)        # add to end       → [1, 2, 3, 4]
lst.insert(0, 0)     # insert at index  → [0, 1, 2, 3, 4]
lst.insert(2, 99)    # insert at index 2 → [0, 1, 99, 2, 3, 4]
```

### Building a list dynamically
```python
my_list = []
for i in range(1, 6):
    my_list.append(i)
print(my_list)   # [1, 2, 3, 4, 5]
```

---

## Removing elements

```python
lst = [1, 2, 3, 4, 5]
del lst[2]         # remove by index → [1, 2, 4, 5]
del lst            # delete the entire list
```

---

## Iterating over a list

```python
my_list = [10, 1, 8, 3, 5]

# By index
for i in range(len(my_list)):
    print(my_list[i])

# By value (cleaner)
for value in my_list:
    print(value)

# Sum all elements
total = 0
for value in my_list:
    total += value
print(total)   # 27
```

---

## Swapping elements

```python
# Python one-liner swap
a, b = b, a

# Reverse a list using swaps
my_list = [10, 1, 8, 3, 5]
length = len(my_list)
for i in range(length // 2):
    my_list[i], my_list[length - i - 1] = my_list[length - i - 1], my_list[i]
print(my_list)   # [5, 3, 8, 1, 10]
```

---

## Nested lists

```python
lst = [1, [2, 3], 4]
print(lst[1])      # [2, 3]
print(lst[1][0])   # 2
print(len(lst))    # 3 — outer list has 3 elements
```

---

## Functions vs Methods

| | Function | Method |
|---|---|---|
| Syntax | `len(lst)` | `lst.append(x)` |
| Belongs to | The whole code | The data (list) |
| Example | `len()`, `del` | `append()`, `insert()` |

### Exercise 2 — Running total
Given `lst = [1, 2, 3, 4, 5]`, build a new list where each element is the cumulative sum.

<details>
<summary>Solution</summary>

```python
lst = [1, 2, 3, 4, 5]
lst_2 = []
add = 0
for number in lst:
    add += number
    lst_2.append(add)
print(lst_2)   # [1, 3, 6, 10, 15]
```
</details>

### Exercise 3 — Hat trick
Given `hat = [1, 2, 3, 4, 5]`:
- Replace the middle element with user input
- Remove the last element
- Print the length

<details>
<summary>Solution</summary>

```python
hat = [1, 2, 3, 4, 5]
hat[2] = int(input("Enter a number: "))
del hat[-1]
print(len(hat))   # 4
```
</details>
