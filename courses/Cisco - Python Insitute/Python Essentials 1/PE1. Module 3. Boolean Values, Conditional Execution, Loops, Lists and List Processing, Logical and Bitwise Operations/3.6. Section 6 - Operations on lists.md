## Lists are references — not copies!

Assigning a list to another variable copies the **reference**, not the content. Both names point to the same list.

```python
list_1 = [1, 2, 3]
list_2 = list_1       # same list in memory!
list_1[0] = 99
print(list_2)         # [99, 2, 3] — list_2 changed too!
```

> ✅ Use slicing to make a real copy.

---

## Slices

Create a **new** copy of all or part of a list.

```python
# Syntax: my_list[start:end]  → elements from start to end-1

lst = [10, 8, 6, 4, 2]

lst[:]      # full copy → [10, 8, 6, 4, 2]
lst[1:3]    # index 1 and 2 → [8, 6]
lst[:3]     # from start to index 2 → [10, 8, 6]
lst[2:]     # from index 2 to end → [6, 4, 2]
lst[1:-1]   # index 1 to second-to-last → [8, 6, 4]
lst[-2:]    # last two → [4, 2]
```

```python
# Safe copy — independent from original
list_1 = [1, 2, 3]
list_2 = list_1[:]    # real copy
list_1[0] = 99
print(list_2)         # [1, 2, 3] — unchanged
```

---

## Deleting slices

```python
lst = [10, 8, 6, 4, 2]
del lst[1:3]      # remove elements at index 1 and 2
print(lst)        # [10, 4, 2]

del lst[:]        # clear the list
print(lst)        # []

del lst           # delete the variable entirely → NameError if accessed
```

## in and not in operators

Check if a value exists in a list.

```python
lst = ["A", "B", 1, 2]

print("A" in lst)       # True
print("C" not in lst)   # True
print(2 not in lst)     # False
```

### Practical: lottery checker
```python
drawn = [5, 11, 9, 42, 3, 49]
bets  = [3, 7, 11, 42, 34, 49]
hits  = 0

for number in bets:
    if number in drawn:
        hits += 1

print(hits)   # 4
```

## Common list patterns

### Find maximum
```python
lst = [17, 3, 11, 5, 1, 9]
largest = lst[0]
for value in lst[1:]:
    if value > largest:
        largest = value
print(largest)   # 17
```

### Find index of element
```python
lst = [1, 2, 3, 4, 5]
to_find = 3
found = False
for i in range(len(lst)):
    if lst[i] == to_find:
        found = True
        break
print("Found at index", i if found else "not found")
```

### Remove duplicates
```python
lst = [1, 2, 4, 4, 1, 4, 2, 6, 3]
unique = []
for item in lst:
    if item not in unique:
        unique.append(item)
print(unique)   # [1, 2, 4, 6, 3]
```

### Exercise 2 — Safe copy
Show the difference between `list_2 = list_1` and `list_2 = list_1[:]`.

<details>
<summary>Solution</summary>

```python
# Reference — both change
list_1 = [1, 2, 3]
list_2 = list_1
list_1[0] = 99
print(list_2)   # [99, 2, 3]

# Copy — independent
list_1 = [1, 2, 3]
list_2 = list_1[:]
list_1[0] = 99
print(list_2)   # [1, 2, 3]
```
</details>

<details>
<summary>Q1 — list_2 = list_1 / list_3 = list_2 / del list_1[0] / del list_2[0]</summary>

```
['C']
```
All three names point to the same list. Two elements deleted → `['C']`.
</details>

<details>
<summary>Q2 — same but del list_2 (not del list_2[0])</summary>

```
['B', 'C']
```
`del list_2` removes the **name** `list_2`, not the list content. `list_3` still points to the list with `'A'` removed by `del list_1[0]`.
</details>

<details>
<summary>Q3 — del list_2[:] instead</summary>

```
[]
```
`del list_2[:]` clears the list content. All three names point to the same (now empty) list.
</details>

<details>
<summary>Q4 — list_2 = list_1[:] / list_3 = list_2[:]</summary>

```
['A', 'B', 'C']
```
Slices create independent copies. Deleting from `list_1` and `list_2` doesn't affect `list_3`.
</details>

<details>
<summary>Q5 — in or not in</summary>

```python
print(1 in my_list)        # True
print("A" not in my_list)  # True  — "A" ≠ "ABC"
print(3 not in my_list)    # True
print(False not in my_list)# False — False == 0, and 0 is not in list, but True is (True==1==1) 
```

> ⚠️ `True in [1, 2]` → `True` because `True == 1`.
</details>
