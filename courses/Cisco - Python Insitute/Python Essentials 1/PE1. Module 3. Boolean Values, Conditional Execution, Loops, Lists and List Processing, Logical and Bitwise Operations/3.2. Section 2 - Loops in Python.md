## while loop

Repeats a block **as long as the condition is True**.

```python
counter = 5
while counter > 0:
    print(counter)
    counter -= 1
# Output: 5 4 3 2 1
```

> ⚠️ The body must change the condition eventually — otherwise you get an **infinite loop**.

```python
while True:
    print("infinite!")   # runs forever — use Ctrl+C to stop
```

---

## for loop + range()

Iterates a fixed number of times using `range()`.

```python
# range(stop)          → 0 to stop-1
# range(start, stop)   → start to stop-1
# range(start, stop, step)

for i in range(5):          # 0 1 2 3 4
    print(i)

for i in range(2, 8):       # 2 3 4 5 6 7
    print(i)

for i in range(0, 10, 2):   # 0 2 4 6 8
    print(i)

for i in range(10, 0, -1):  # 10 9 8 ... 1 (countdown)
    print(i)
```

> `range()` only accepts **integers**. Empty range = loop never runs.

---

## break — exit the loop

Stops the loop immediately regardless of the condition.

```python
for i in range(10):
    if i == 5:
        break
    print(i)
# Output: 0 1 2 3 4
```

---

## continue — skip current iteration

Jumps to the next iteration without finishing the current one.

```python
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)
# Output: 1 3 5 7 9  (odd numbers only)
```

---

## else clause

Runs **once** after the loop finishes — but **not** if `break` was used.

```python
for i in range(3):
    print(i)
else:
    print("done")   # always runs if no break
# Output: 0 1 2 done

for i in range(3):
    if i == 1:
        break
else:
    print("done")   # NOT printed — loop was broken
```

## Common patterns

### Find largest number (sentinel -1 to stop)
```python
largest = -999999999
number = int(input("Enter number (-1 to stop): "))
while number != -1:
    if number > largest:
        largest = number
    number = int(input("Enter number (-1 to stop): "))
print("Largest:", largest)
```

### Count odd/even numbers
```python
odd = even = 0
number = int(input("Enter number (0 to stop): "))
while number != 0:
    if number % 2 == 1:
        odd += 1
    else:
        even += 1
    number = int(input("Enter number (0 to stop): "))
print("Odd:", odd, "Even:", even)
```

### Vowel eater
```python
word = input("Enter word: ").upper()
result = ""
for letter in word:
    if letter in "AEIOU":
        continue
    result += letter
print(result)
```

### Collatz sequence
```python
c = int(input("Enter a number: "))
steps = 0
while c != 1:
    if c % 2 == 0:
        c //= 2
    else:
        c = 3 * c + 1
    print(c)
    steps += 1
print("steps =", steps)
```

## Exercises

### Exercise 1 — for + range
Print all odd numbers from 1 to 10.

<details>
<summary>Solution</summary>

```python
for i in range(1, 11):
    if i % 2 != 0:
        print(i)
```
</details>

### Exercise 2 — while
Same as above but with a `while` loop.

<details>
<summary>Solution</summary>

```python
x = 1
while x < 11:
    if x % 2 != 0:
        print(x)
    x += 1
```
</details>

### Exercise 3 — break
Iterate over `"john.smith@pythoninstitute.org"` and print characters before `@`.

<details>
<summary>Solution</summary>

```python
for ch in "john.smith@pythoninstitute.org":
    if ch == "@":
        break
    print(ch, end="")
```
</details>

---

### Exercise 4 — continue
Iterate over `"0165031806510"` and replace every `0` with `x`.

<details>
<summary>Solution</summary>

```python
for digit in "0165031806510":
    if digit == "0":
        print("x", end="")
        continue
    print(digit, end="")
```
</details>
