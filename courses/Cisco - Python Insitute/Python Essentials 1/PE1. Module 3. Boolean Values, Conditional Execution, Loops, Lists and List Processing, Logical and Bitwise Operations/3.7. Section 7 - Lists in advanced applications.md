## The input() function

Reads data entered by the user from the console and returns it as a **string**.

```python
name = input("Enter your name: ")
print("Hello, " + name)
```

> ⚠️ `input()` **always returns a string** — even if the user types a number.

---

## Type casting

Convert the string input to the type you need:

| Function | Converts to | Example |
|---|---|---|
| `int()` | Integer | `int("42")` → `42` |
| `float()` | Float | `float("3.14")` → `3.14` |
| `str()` | String | `str(42)` → `"42"` |

```python
# ❌ This crashes — can't do math on a string
x = input("Enter a number: ")
print(x ** 2)   # TypeError!

# ✅ Convert first
x = float(input("Enter a number: "))
print(x ** 2)
```

### Chaining input and conversion

```python
a = float(input("Leg a: "))
b = float(input("Leg b: "))
print("Hypotenuse:", (a ** 2 + b ** 2) ** 0.5)
```

## String operators

### `+` — concatenation

Joins two strings together. Both operands **must** be strings.

```python
first = "Monty"
last = "Python"
print(first + " " + last)   # Monty Python

# ❌ Cannot mix string and int
print("Value: " + 42)       # TypeError
# ✅ Convert first
print("Value: " + str(42))  # Value: 42
```

### `*` — replication

Repeats a string a given number of times.

```python
print("ha" * 3)      # hahaha
print("-" * 20)      # --------------------
print(3 * "ab")      # ababab
print("x" * 0)       # (empty string)
```

---

## str() — number to string

```python
c = (3.0 ** 2 + 4.0 ** 2) ** 0.5
print("Hypotenuse: " + str(c))   # Hypotenuse: 5.0
```

### Exercise 3 — End time calculator
Ask for a start time (hours and minutes) and a duration in minutes. Print the end time using `%` to wrap around 60/24.

<details>
<summary>Solution</summary>

```python
hour = int(input("Start hour: "))
minute = int(input("Start minute: "))
duration = int(input("Duration (minutes): "))

total_minutes = hour * 60 + minute + duration
end_hour = (total_minutes // 60) % 24
end_minute = total_minutes % 60

print(str(end_hour) + ":" + str(end_minute))
```
</details>
